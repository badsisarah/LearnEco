# 🌱 LearnEco – Application éducative écologique

## 📌 Description
LearnEco est une application desktop éducative conçue dans le cadre d’un projet entrepreneurial de micro-startup.

L’objectif du projet est de proposer une plateforme interactive de sensibilisation à l’écologie, combinant une interface utilisateur moderne, une logique métier en Java et une gestion des données avec SQLite.

Le projet s’inscrit dans une réflexion produit incluant l’analyse des besoins utilisateurs et la mise en place d’un modèle freemium.

---

## 🎯 Objectifs du projet
- Sensibiliser les utilisateurs aux enjeux écologiques
- Proposer une expérience éducative interactive sur application desktop
- Concevoir une architecture claire séparant interface, logique métier et données
- Expérimenter une approche produit avec un modèle freemium

---

## ⚙️ Fonctionnalités
- Interface graphique développée avec JavaFX
- Gestion de la logique applicative en Java
- Persistance des données avec SQLite
- Connexion à la base via JDBC
- Organisation du projet en frontend et backend
- Réflexion autour d’un modèle freemium

---

## 🧱 Architecture du projet
- **Frontend** : interface utilisateur en JavaFX
- **Backend** : logique métier et traitements en Java
- **Base de données** : SQLite
- **Accès aux données** : JDBC

---

## 🛠️ Technologies utilisées
- Java
- JavaFX
- SQLite
- JDBC

---

## 🧠 Compétences développées
- Développement d’application desktop
- Conception d’interface graphique en JavaFX
- Structuration frontend / backend
- Modélisation et gestion de base de données
- Connexion Java ↔ SQLite avec JDBC
- Analyse des besoins utilisateurs
- Réflexion produit et modèle freemium

---

## 🚀 Contexte entrepreneurial
Ce projet a été réalisé dans le cadre du développement d’une micro-startup.

Il ne s’agit pas uniquement d’un projet technique, mais aussi d’une démarche de conception produit intégrant :
- l’identification des besoins utilisateurs,
- la définition d’une proposition de valeur,
- la réflexion sur une offre freemium.
